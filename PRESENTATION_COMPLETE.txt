🚀 API REST (20+ ENDPOINTS)
═══════════════════════════════════════════════════════════════════════════════

FOURNISSEURS:
  GET    /api/suppliers                  → Lister tous
  GET    /api/suppliers/:id              → Détails fournisseur
  POST   /api/suppliers                  → Créer fournisseur
  PUT    /api/suppliers/:id              → Mettre à jour

PRODUITS:
  GET    /api/products                   → Lister (paginage)
  GET    /api/products/:id/details       → Détails + variantes + fournisseurs
  POST   /api/products                   → Créer produit
  POST   /api/products/:id/variants      → Ajouter variante

FOURNISSEUR-PRODUITS:
  GET    /api/supplier-products          → Lister avec filtres
  POST   /api/supplier-products          → Ajouter mapping
  PUT    /api/supplier-products/:id      → Mettre à jour prix/stock

SYNCHRONISATION WORDPRESS:
  POST   /api/sync/product/:id           → Créer produit WooCommerce
  PUT    /api/sync/stock/:wpProductId    → Mettre à jour stock
  GET    /api/orders/recent              → Récupérer commandes

DASHBOARD:
  GET    /api/dashboard/inventory        → Vue d'ensemble stock
  GET    /api/dashboard/supplier-margins → Analyser marges

SANTÉ:
  GET    /api/health                     → Statut API + MongoDB
