### Plugin recommandé : WooCommerce Dropshipping
- Synchronisation automatique des stocks
- Mise à jour des prix en temps réel
- Notifications de rupture de stock

### Configuration des produits WP
```
Titre : [Brand] + [Nom Produit]
Description : Avantages + Âge recommandé + Spécifications
Catégorie : Hygiène/Soins/Bain/Repas/Déplacement
Balise : Par âge (0-3 mois, 3-6 mois, etc.)
```

## 8. NOTES IMPORTANTES

⚠️ **Vérifier avant commande** :
- Disponibilité en stock (certains fournisseurs pratiquent le backorder)
- Délais actualisés (peuvent varier en haute saison)
- Frais de port spécifiques pour ta région
- Conditions de retour

✅ **Meilleures pratiques** :
- Démarrer avec 2-3 fournisseurs principaux
- Tester la qualité avec petit volume initial
- Comparer prix + délai + qualité
- Négocier à partir de 3000€/mois de volume
- Garder 1 fournisseur backup pour ruptures

💡 **Conseil supplémentaire** :
Pour les kits, privilégier fournisseurs différents par catégorie pour optimiser les délais d'assemblage et les coûts de port.
